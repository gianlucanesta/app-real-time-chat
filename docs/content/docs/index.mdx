**Ephemeral** is a WhatsApp-style real-time chat application where all messages auto-expire after 24 hours. Built as a university project for the Web Development course (COMP-2002), it demonstrates modern full-stack engineering patterns including polyglot persistence, real-time communication, and peer-to-peer calling.

## Tech Stack

| Layer | Technology |
|---|---|
| **Frontend** | React 19, TypeScript, Vite, TanStack Router, Tailwind CSS 4 |
| **Backend** | Node.js, Express, TypeScript |
| **Real-Time** | Socket.io (WebSocket + polling fallback) |
| **Calls** | WebRTC (peer-to-peer voice/video + screen sharing) |
| **Databases** | PostgreSQL (users, contacts), MongoDB (messages, stories), Redis (TTL events) |
| **Storage** | Cloudinary (media uploads) |
| **Auth** | JWT (access + refresh tokens), OAuth2 (Google, Facebook) |
