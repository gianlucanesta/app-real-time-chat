## Prerequisites

- **Node.js** ≥ 18
- **PostgreSQL** (users & contacts)
- **MongoDB** (messages & stories)
- **Redis** (optional — TTL keyspace events)

## Clone the Repository

```bash
git clone https://github.com/gianlucanesta/app-real-time-chat.git
cd app-real-time-chat
```

## Backend Setup

```bash
cd server
npm install
cp .env.example .env   # fill in values — see Environment Variables
npm run dev
```

The API server starts on **port 3001** by default.

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

The Vite dev server starts on **port 5173** and proxies API requests to the backend.
