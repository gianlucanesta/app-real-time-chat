Ephemeral uses **Vitest** for both client and server unit tests, with plans for Playwright end-to-end testing.

## Test Stack

| Tool | Usage |
|---|---|
| **Vitest** | Test runner (client v4, server v2) |
| **@testing-library/react** | Component rendering + assertions |
| **jsdom** | Browser environment simulation |
| **supertest** | HTTP endpoint testing (server) |

## Running Tests

### Client

```bash
cd client
npm test          # single run
npm run test:watch # watch mode
```

### Server

```bash
cd server
npm test
```

### Full Suite

From the project root:

```bash
cd client && npm test && cd ../server && npm test
```

## CI/CD

Tests run automatically on every push and pull request via **GitHub Actions**:

```yaml
# .github/workflows/ci.yml
jobs:
  client:
    - npm ci
    - tsc --noEmit
    - npm test
  server:
    - npm ci
    - tsc --noEmit
    - npm test
```

Both jobs must pass before merging.
